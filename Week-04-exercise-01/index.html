What I "Vandalized"

For this exercise, I used browser developer tools to edit a well-known website homepage. I changed the main text on the page to say “Welcome to Slimeto University.”

I also modified the CSS styles by changing the color of the text to green. This made the page look different from the original version and gave it a funny, custom look.

My edits were interesting because they made the website look like it had been changed into my own fake university page. The “Welcome to Slimeto University” text made it feel creative and personal.

Some changes worked right away because I edited the correct HTML text and CSS properties in the developer tools. Other changes were harder because some parts of the webpage had styles coming from different CSS rules, so I had to click around and find the right element.

I chose these changes because they were simple but noticeable. Changing the text and the color helped me understand how HTML controls the content of a webpage and CSS controls how the webpage looks.

This exercise helped me understand how HTML and CSS work together. The HTML gave the page its words and structure, while the CSS controlled the colors, size, and style.

I now have a better sense of how developer tools can be used when building my own webpages. I can use dev tools to test changes, fix problems, and experiment with styles before editing my actual code.
